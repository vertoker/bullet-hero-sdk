Bullet Hero allows to create UGC (user-generated content) and it must be licensing.
Licensing is simular to Geometry Dash, but "one music track" replace to ALL external resources for level

## Level license

All user levels distribute under license **CC BY-NC**. It means
- **CC** - Creative Commons license
- **BY** - Attribution required (credits the authors)
- **NC** - Non‑commercial use only

This license covers the level's own original content - its layout, timing/keyframes,
custom code and the creative work of arranging everything into a finished level.
**It does NOT automatically relicense the external resources embedded in it.**

Each external resource keeps its own license/permission status, tracked separately
per-resource (see "External resources" below) - CC BY-NC on the level is a wrapper
around resources that are each, individually, already safe to redistribute that way.
A level only qualifies for CC BY-NC distribution if EVERY external resource in it
satisfies Option A or Option B below; if even one doesn't, the level can't be
distributed through Bullet Hero official web services (aka. official server, next is OWS).

I will not prevent free distribution through unofficial sources, but I categorically
against the commercial use of levels.

## External resources

External resources is a wide range of data, it includes
- Music/Audio (`.mp3`, `.wav`, `.ogg`)
- Textures/Images (`.png`, `.jpg`, `.jpeg`)
- Fonts (`.tff``)
- Texts/Bytes (whatever you want)

Each external resource must satisfy one of these two options.

### Option A - open license

The resource is available under a license that has **same license or more free** than
CC BY-NC (see the allowed-license list below). The resource keeps its OWN original
license here - it isn't "converted" into CC BY-NC, it already permits the kind of public
non-commercial redistribution CC BY-NC requires, so including it in a CC BY-NC level is
consistent. The resource's original license and source must be recorded in the level's
metadata file (same requirement SIL OFL already has - this applies to every resource,
not just fonts), so anyone who downloads the level can see exactly under what terms
each piece was used.

### Option B - author's direct permission

Get from author/artist "permission to use" specially for this game. A private "sure, go
ahead" is not enough by itself - a level marked CC BY-NC is redistributed publicly, so
the permission has to explicitly cover that, not just your own personal use. Valid
permission must grant all of:
1. The right to use the resource inside a level in Bullet Hero.
2. The right for the resulting level (including this resource) to be freely
   redistributed **non-commercially** by third parties through Bullet Hero's services
   (OWS, Steam Workshop, NOWS), with attribution to the author.
3. Explicitly non-commercial use only.

This can be anything that moderators can check: mail, direct message or public statement
(like Twitter/X). Use the template below as a starting point, and keep the author's reply
as your proof.

#### Permission request template

Fill in the placeholders and adjust the tone as needed, but keep points 1-3 intact -
that's what actually makes the permission valid for CC BY-NC distribution.

```
Subject: Permission to use "<TRACK/RESOURCE NAME>" in a Bullet Hero level

Hi <AUTHOR NAME>,

My name is <YOUR NAME / HANDLE>, I'm creating a level for Bullet Hero (a rhythm/
bullet-hell game - <link to game/project if available>) and I'd love to use your
work "<TRACK/RESOURCE NAME>" (<link to the original>) in it.

Specifically, I'm asking your permission to:
1. Use "<TRACK/RESOURCE NAME>" inside my Bullet Hero level.
2. Allow the level (including your work) to be shared/redistributed for FREE and
   NON-COMMERCIALLY by anyone who downloads it, through Bullet Hero's official and
   unofficial services (including its official server, Steam Workshop, and any
   community-run servers).
3. Credit you as "<HOW YOU WANT TO BE CREDITED>" wherever the level shows attribution.

To be clear, this permission is for non-commercial use only - the level will not be
sold, and no one (including me) will make money from it or from your work through it.

If you're fine with this, a simple reply confirming points 1-3 is all I need - I'll
keep this email/message as proof of permission. If you'd rather I credit you
differently, or you want to set any other condition, just let me know.

Thanks a lot either way!
<YOUR NAME / HANDLE>
```

## External services

Game allows to use several external services, which is
- Official Web Services (OWS) - default server, has on all platforms by default
- Steam Workshop - has on PC by default
- Non-official web services (NOWS) - user allows to create any web services and use it as he wants.
All tools for custom server is open source and under MIT license

## Allowed licenses (Option A)

"Allowed" external resources must have this typical licenses (same or more free)
- Creative Commons Family
  - CC BY-NC (Attribution-NonCommercial) - same to level
  - CC BY (Attribution) - more free than CC BY-NC
  - CC0 (Public Domain) - best option, use this license if you can
- GPL, Apache, MIT - specifically for text/code, but they are more free than CC BY-NC
- SIL OFL (Open Font License) - requires to specity it in metadata file

Typical "not allowed" licenses
- Proprietary ("All Rights Reserved") - obviously not allowed
- Creative Commons Family
  - CC BY-SA (Attribution-ShareAlike) - technically is allowed,
but require to change level license to CC BY-NC-SA (which is not allowed for OWS), so answer is no
  - CC BY-ND (Attribution-NoDerivation) - technically is allowed,
but require block level modification (which game is not supported), so answer is no
  - CC BY-NC-SA (Attribution-NonCommercial-ShareAlike)
  - CC BY-NC-ND (Attribution-NonCommercial-NoDerivation)
